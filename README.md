# Inter-VLAN Security Lab

## Objective
Implement VLAN segmentation with secure communication between departments.

## Current Topology
2 Switches
1 Router
4 PC's

## VLANs
VLAN 10: Department A = HR
VLAN 20: Department B = Finance 
VLAN 30: IT
VLAN 40: Guest

## Planned Security Controls
- Inter-VLAN routing
- ACL restrictions
- SSH management
- Password Encryption

## Planned Results 
- IT can access all departments
- Guest network isolated
- Unauthorised access is blocked
